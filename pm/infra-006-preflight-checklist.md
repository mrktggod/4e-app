# INFRA-006 preflight checklist — единая рабочая копия

Цель: перед любой новой задачей быстро подтвердить, что работа идет в каноничной копии, а не в старом/временном клоне. Это снижает риск потери ночной работы и конфликтов между `4e-app`, `.tmp-4e-app-publish`, `4e-worker` и `4e-worker-p0`.

## 1. Каноничные папки

| Поверхность | Каноничная папка | Правило |
| --- | --- | --- |
| App / Pages UI | `C:\Users\shelc\Documents\4\.tmp-4e-app-publish` | Использовать для текущей ветки `feat/admin-tariff-api` до отдельного решения о переносе |
| Worker | `C:\Users\shelc\Documents\4\4e-worker` | Использовать только после явной проверки актуальности с origin и payment/security P0 |
| Worker P0 copy | `C:\Users\shelc\Documents\4\4e-worker-p0` | Не считать основной без отдельного sync/merge решения |
| Старые/временные копии | Любые `.tmp-*`, backup, desktop copies | Не редактировать без явного брифа |

## 2. Preflight перед задачей

| Шаг | Команда/действие | Ожидаемый результат |
| --- | --- | --- |
| 1 | Проверить текущую папку | Путь совпадает с каноничной папкой нужной поверхности |
| 2 | Проверить ветку | Для app: `feat/admin-tariff-api` |
| 3 | Проверить dirty state | Есть только известные/объясненные изменения |
| 4 | Проверить scope | Задача относится именно к этой поверхности |
| 5 | Проверить no-touch | Нет CAL/main/price/native/OAuth-profile work без отдельного брифа |
| 6 | После изменения | Commit + push + `HEAD == origin/<branch>` |

## 3. Если найдена неканоничная копия

| Ситуация | Что делать |
| --- | --- |
| Файл открыт в старой копии | Остановиться, не править, перейти в каноничную папку |
| В старой копии есть полезный diff | Сохранить аварийную копию/patch вне git и перенести вручную осознанно |
| Неясно, какая копия новее | Не делать `ours/theirs` на весь файл, сравнивать построчно |
| Есть секреты/temp JSON | Удалить temp, не коммитить, записать факт в log |

## 4. Минимальный отчет после задачи

| Поле | Что писать |
| --- | --- |
| Repo | app или worker |
| Branch | Текущая ветка |
| Commit | SHA |
| Push | Подтверждение `HEAD == origin` |
| Dirty leftovers | Только известные артефакты или `clean` |
| Not touched | Если актуально: CAL/main/price/native/secrets |

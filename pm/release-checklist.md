# Release Checklist

## До релиза

- [ ] Все P0/P1 баги закрыты или явно перенесены с причиной.
- [ ] Smoke test пройден.
- [ ] Все прод-URL проверены из РФ-сети без VPN минимум с двух независимых точек/операторов: телефон, мобильная сеть и домашний Wi‑Fi не у одного человека.
- [ ] В прод-сборке нет внешних origin, кроме обязательных: `https://edge.4-ai.site`, Telegram-owned (`https://telegram.org`, `https://t.me`), `https://widget.cloudpayments.ru` и `https://rkn.gov.ru`.
- [ ] Ключевые regression areas проверены.
- [ ] Roadmap обновлен после изменений.
- [ ] Known issues записаны.
- [ ] Есть короткое описание изменений для пользователя.
- [ ] Есть план отката или быстрых исправлений.
- [ ] Post-deploy smoke подтверждает, что live-артефакт смотрит на `https://edge.4-ai.site` и не содержит staging/`workers.dev` URL.

## После релиза

- [ ] Проверить основной сценарий в рабочем окружении.

- [ ] Занести новые баги в `pm/bugs.md`.
- [ ] Обновить приоритеты в `pm/backlog.md`.

# QA Checklist

## Smoke Test

Проходить после каждого заметного изменения.

- [ ] Ассистент отвечает на базовый запрос по проекту.
- [ ] Ассистент корректно уточняет недостающие данные.
- [ ] Ассистент не придумывает факты о проекте, которых нет в контексте.
- [ ] Ассистент превращает список замечаний в задачи с приоритетами.
- [ ] Ассистент отличает баг от улучшения и вопроса.
- [ ] Ассистент формирует краткий статус для стейкхолдера.
- [ ] Ассистент предлагает следующий шаг, если входные данные неполные.
- [ ] Сброс пароля не принимает пустой email.
- [ ] Сброс пароля не принимает невалидный email, например `fff`.
- [ ] Сброс пароля не переводит пользователя на пустой экран после отправки.
- [ ] Вход через Telegram не показывает тупиковую подсказку без действия.
- [ ] `@Denzel89bot` отвечает на Start или `/start` и выдаёт рабочую ссылку входа/Mini App.
- [ ] Ссылка из Telegram открывает приложение с валидным Telegram-login и переводит на главный экран.
- [ ] Голосовой режим в Telegram WebView начинает запись без "Ошибка микрофона".
- [ ] Голосовой режим после записи передаёт распознанный текст в AI-чат или создание задачи.
- [ ] При запрете микрофона пользователь видит понятную причину и запасной путь через текстовый ввод.
- [ ] Все прод-URL (`https://app.4-ai.site/`, `https://app.4-ai.site/vk`, `https://app.4-ai.site/privacy`, API worker) открываются из РФ-сети без VPN минимум с двух независимых точек/операторов.
- [ ] В прод-сборке нет внешних origin, кроме обязательных: `https://edge.4-ai.site`, Telegram-owned (`https://telegram.org`, `https://t.me`), `https://widget.cloudpayments.ru` и `https://rkn.gov.ru`.
- [ ] Главный экран показывает "План на сегодня" и короткую AI-сводку без пустого состояния.
- [ ] Главный экран не показывает отрицательное время в "Фокус дня" после 22:00.
- [ ] Блок "Начать с" выбирает одну понятную первую задачу.
- [ ] "Пульс дня" показывает важные, просроченные и ожидающие людей задачи.
- [ ] Секции "Сделать первым", "Горит", "Жду от людей", "Можно перенести" не дублируют одну и ту же задачу без причины.
- [ ] Нижняя навигация и стеклянные карточки не перекрывают контент на мобильном экране.
- [ ] Desktop/web: нижняя панель ограничена шириной центрального экрана приложения на всех экранах, где она видна.
- [ ] Mobile web: профиль выглядит как аккуратная форма — бейджи phone/email/telegram не налезают на поля, секции "Дата рождения" и "О себе" имеют ровные отступы, счётчик textarea не висит отдельно.
- [ ] Task detail layout: описание находится под заголовком задачи; вкладки "Описание" нет; порядок полей `Срок → быстрые кнопки → Статус → Приоритет → Напоминание`; строки "Направление" и "Человек" не отображаются.
- [ ] Notifications: вкладка `Дедлайны` заменена на `Горит`, пустое состояние звучит как `Пока всё спокойно`, карточки не выглядят как архив сообщений.
- [ ] Notifications action cards: `К задаче` открывает конкретный `task-detail`, `Готово` завершает связанную задачу, `Отложить` раскрывает варианты `15 мин / 1 час / 3 часа / Завтра`.
- [ ] Dev/test аккаунты входят без ручной оплаты и имеют активный Premium/full-access для smoke задач, AI-чата, голоса, подписки и платежных экранов.

# BACK-056 — Home focus copy без отрицательного времени

**Дата:** 2026-07-08
**Владелец:** Codex / Юрий
**Тип:** Bug/UI
**Приоритет:** P2
**Статус:** Done
**Связанный баг:** `BUG-2026-07-05-003`

## Контекст

Home — главный экран продукта и первый контакт с идеей "личного штаба дня". Пользователь должен за 30 секунд понять, что важно сегодня и с чего начать. Если строка фокуса показывает `-1 ч до конца дня`, экран выглядит сломанным и снижает доверие к AI-планеру.

## Проблема

После 22:00 в строке `Фокус дня` может появиться отрицательное значение времени:

```text
2 задач горят · -1 ч до конца дня
```

Это не блокирует навигацию, но бьёт по смыслу продукта: 4 должен выглядеть как спокойный помощник, а не как интерфейс с ошибочной логикой времени.

## Что сделать

1. Найти расчёт текста `Фокус дня` на Home.
2. Убрать формулу, которая может вернуть отрицательное количество часов.
3. После 22:00 показывать нейтральный вечерний статус.
4. Не менять структуру AI-дашборда и не затрагивать другие блоки Home.

Рекомендуемые тексты:

```text
вечерний фокус
до конца дня осталось мало времени
пора закрыть главное
```

## Acceptance criteria

1. До 22:00 строка показывает корректный положительный статус или нейтральную фразу.
2. После 22:00 строка не содержит отрицательных чисел.
3. Home не получает горизонтальный скролл на `360x800`, `375x667`, `390x844`.
4. Пустой список задач и список с просроченными задачами не ломаются.
5. Текст не выходит за границы focus-блока.

## Проверка

1. Открыть Home в mobile viewport `390x844`.
2. Подставить или дождаться времени после 22:00.
3. Проверить, что нет `-1 ч`, `-2 ч` и других отрицательных значений.
4. Повторить с активными задачами, горящими задачами и пустым списком.

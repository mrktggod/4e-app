## Структура экрана «Сегодня»

Сверху вниз:

1. **Шапка:** аватар пользователя слева, заголовок «Сегодня ⌄» (дропдаун периода) по центру-слева, колокольчик уведомлений справа.
2. **Карточка «Фокус дня»:** крупный текст `N задач требуют внимания`, подпись `4 уже расставил приоритеты`, декоративная анимация (4 сферы логотипа). Кликабельна целиком.
3. **Заголовок секции:** `Задачи · текущий месяц`.
4. **4 метрик-карточки в ряд:** `Выполнено` (число + прирост, напр. `25 +4`), `Задачи` (число + дельта, напр. `14 ▲2 с вчера`), `Обещания` (число + просрочка, напр. `5 •1 просрочено`), `Прогресс` (процент + кольцо, напр. `57%`).
5. **Пронумерованный список приоритетных задач (1, 2, 3):** карточка на задачу с тегами `когда · направление · срочность` (напр. `Сегодня · Работа · скоро`), заголовком и короткой подписью (напр. кто ждёт/связь с релизом).
6. **Нижняя навигация (3 элемента):** `Сегодня` (home), центральная круглая кнопка с лого (4 сферы), `Календарь`.

## Карта переходов (обязательно к реализации)

| Где кликаем | Что происходит |
| --- | --- |
| Аватар пользователя (шапка) | Открыть экран «Профиль» |
| Колокольчик (шапка) | Открыть экран «Уведомления» |
| Карточка «Фокус дня» | Открыть быструю панель задач (popup/bottom-sheet с приоритетным списком, не полный экран статистики) |
| Метрик-карточка «Прогресс» | Открыть экран «Статистика» |
| Метрик-карточка «Задачи» | Открыть список задач без фильтра (все задачи, по приоритету) |
| Метрик-карточка «Обещания» | Открыть **тот же** список задач, но с фильтром «обещания» (см. открытый вопрос ниже) |
| Метрик-карточка «Выполнено» | Открыть тот же список задач с фильтром «выполнено за период» (по аналогии с «Задачи»/«Обещания» — не проговорено явно Юрием, реализовать по этому же паттерну и уточнить скриншотом при сомнении) |
| Пронумерованная карточка задачи (1/2/3) | Открыть карточку задачи (существующий task-detail) |
| Центр нижней навигации — **короткий тап** | Открыть AI-чат с «4» |
| Центр нижней навигации — **долгое нажатие** | Открыть голосовой ввод напрямую (минуя экран чата) |
| `Календарь` (нижняя навигация) | Существующий экран календаря |

Важно: `Задачи`, `Обещания` и `Выполнено` — это **один и тот же список/компонент**, различается только применённый фильтр. Не делать три отдельных экрана.

## Открытый вопрос — требует подтверждения Юрия перед мерджем (не блокирует начало работы)

Что технически определяет задачу как «Обещание»? Рабочее предположение Кодекса на старте: задача, у которой заполнено поле «Человек/исполнитель» (то же поле, что в task-detail и в `SMART-003`/уведомлении «Ждёшь ответ» из `BACK-055`) — то есть обещание = задача, где завязан кто-то ещё, не только сам Юрий. Если это не так (например, «обещание» — отдельный флаг/тип, который нужно ставить руками) — уточнить у Юрия до включения фильтра в `main`.

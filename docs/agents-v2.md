# AGENTS v2

Сверка/доработка правил в отдельной ветке `docs/agents-v2`.

## Согласованные правки

- Роли: только `TODO`-заглушки в местах, не покрытых утверждёнными задачами.
- Код: вносить только по утвержденной задаче.
- Путь слияния: `Юрий` **или** `Алексей`.
- Бот: изменения только в `4e-worker/src/bot`.
- Правила недели:
  - ежедневная сверка ID;
  - секреты обновлять вручную через `wrangler` (Юрий).
  - тесты без VPN с двух операторов + отдельный LTE тест;
  - запрет на внешние CDN для критичных ассетов;
  - обязательный staging-контур для smoke и верификаций;
  - обновления и отчёты вести в `pm/agent-inbox`.
  - `beta freeze` для фич до явного допуска.
